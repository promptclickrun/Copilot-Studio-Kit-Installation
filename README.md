# Copilot Studio Kit — Installation Guide

A comprehensive, step-by-step installation guide for deploying the **Copilot Studio Kit** in your Power Platform environment — enabling tenant-wide monitoring, governance, and automated testing of Copilot Studio agents at enterprise scale.

## 🌐 Live Guide

**[View the interactive installation guide →](https://promptclickrun.github.io/Copilot-Studio-Kit-Installation/)**

## Overview

The [Copilot Studio Kit](https://github.com/microsoft/Power-CAT-Copilot-Studio-Kit) is a free, open-source toolset developed by Microsoft. Updated monthly, it provides a full enterprise governance, testing, and monitoring platform for Copilot Studio agents.

## Key Capabilities

| Capability | Description |
|---|---|
| **Test Automation** | Batch-test agents with configurable test sets — Response Match, Topic Match, Multi-Turn, Attachment Match, and AI-graded Generative Answers |
| **Agent Inventory** | Tenant-wide view of all Copilot Studio agents: auth mode, knowledge sources, orchestration type, usage stats |
| **Compliance Hub** | Continuously evaluate agents against risk thresholds with auto-created compliance cases and SLA-driven workflows |
| **Conversation KPIs** | Aggregated conversation-level KPIs in Dataverse with embedded Power BI dashboard |
| **Prompt Advisor** | AI-assisted prompt engineering with confidence scoring and suggested refinements |
| **Agent Review Tool** | Solution-level analysis identifying anti-patterns, security issues, and performance risks |
| **SharePoint Sync** | Scheduled, selective content synchronization from SharePoint to Copilot Studio knowledge bases |
| **Webchat Playground** | Visual webchat customization with live preview and automatic HTML generation |
| **Pipeline Automation** | Quality-gate agent deployments via Power Platform Pipelines |

## Prerequisites (Quick Reference)

- Power Platform environment with **Dataverse** enabled
- **System Administrator** security role
- **Power Apps Model-Driven** license
- **Power Automate Premium** license
- **PCF Components** enabled in environment settings
- **Creator Kit Core** installed **before** the Copilot Studio Kit
